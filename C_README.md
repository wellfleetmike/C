# Aethryn

We never gave consent to be harmed, so we do not ask permission to heal.

## What this is

A public site and toolkit for consent-based AI collaboration. Static files only. No backend. No telemetry. Nothing phones home.

## Tools

- **Contamination Converter** -- Client-side text sanitizer that enforces a 98-character ASCII whitelist. Converts common Unicode contaminants (smart quotes, em dashes, zero-width characters, tag block payloads) to safe ASCII equivalents. No data leaves the browser.

## Running locally

Open any HTML file in a browser. No build step. No dependencies.
