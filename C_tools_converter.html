ABOUT

The Aethryn Contamination Converter is a client-side text sanitizer. No data leaves your browser. There is no server, no API call, no telemetry. Paste contaminated text in, get clean ASCII out.

This tool enforces a 98-character whitelist -- the same whitelist used by the sovereign-editor toolkit, the contamination scanner, and the GaslitAI detection suite. Any character outside the whitelist is either converted to its ASCII equivalent or stripped entirely.

The threat model: AI-generated text carries hidden Unicode payloads -- zero-width characters, tag block steganography, variation selectors, homoglyphs, private use area insertions, and non-standard whitespace. These characters are invisible to humans but readable by parsers, and can carry prompt injections, tracking data, or encoded messages through copy-paste operations.
